INSOMNIAC CIA INTERFACING TO PMS SOFTWARE

The INSOMNIAC CIA can interface with PMS software either through API calls or via flat text files.

The INSOMNIAC CIA uses a program called FileWatcher to either emulate the file format of a PTI system or DigiGate system.

The FileWatcher must be configured on the computer that the PMS software is transmitting the flat file.

FINDING KEYPAD UNIQUE ID

Open Tech Alliance keypads all have an unique ID's attached to them in order to be able to properly identify keypads, despite the name they may be assigned. This is often required when troubleshooting access zones.

In order to get such UID, the keypad must be powered on. They will need to press numeric 0, the asterisk symbol * and the pound sign simultaneously. This could potentially take multiple tries. Once the service screen appears, the UID will be the 3rd set of 4 characters (from left to right).
